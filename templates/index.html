📖 什么是黄金ETF？

💰 黄金ETF是什么？

黄金ETF(Exchange Traded Fund)是一种在交易所上市交易的开放式基金，以黄金为基础资产，追踪黄金价格波动。投资者可以像买卖股票一样交易黄金ETF。

📈 518880 华夏黄金ETF

国内规模最大的黄金ETF之一，主要投资于上海黄金交易所的黄金现货合约，为投资者提供便捷的黄金投资渠道。

⚠️ 投资风险

黄金ETF受金价波动影响，还包括市场风险、流动性风险等。本系统仅供参考，不构成投资建议，投资需谨慎。

📚 技术指标说明

移动平均线 (MA)

  • MA5: 5日收盘价平均值，反映短期趋势
  • MA10: 10日收盘价平均值，反映中期趋势
  • MA20: 20日收盘价平均值，反映中长期趋势
  • ✓ 多头排列: MA5 > MA10 > MA20，看涨信号
  • ✗ 空头排列: MA5 < MA10 < MA20，看跌信号
  • ⚡ 价格上穿均线: 短期可能反弹
  • ⚡ 价格下穿均线: 短期可能调整

MACD (指数平滑异同移动平均线)

  • MACD线: 短期EMA减长期EMA，反映价格动量
  • Signal线: MACD的9日EMA，作为触发线
  • Histogram: MACD与Signal的差值柱状图
  • ✓ 金叉: MACD上穿Signal，买入信号
  • ✗ 死叉: MACD下穿Signal，卖出信号
  • ✓ 零轴上方: 多方市场
  • ✗ 零轴下方: 空方市场
  • ⚡ Histogram放大: 趋势加强
  • ⚡ Histogram收缩: 趋势减弱

KDJ (随机指标)

  • K值: 快速随机指标，反映近期价格位置
  • D值: K值的平滑平均值，更稳定
  • J值: 3×K - 2×D，对价格变化最敏感
  • ⚠️ J值 > 80: 超买区域，警惕回调风险
  • ⚠️ J值 < 20: 超卖区域，关注反弹机会
  • ✓ 金叉: K上穿D，买入信号
  • ✗ 死叉: K下穿D，卖出信号
  • 取值范围: 0-100，50为多空分界

RSI (相对强弱指数)

  • 计算原理: 衡量价格涨跌的相对强度，取值0-100
  • ⚠️ RSI > 70: 超买区域，市场可能过热
  • ⚠️ RSI < 30: 超卖区域，市场可能超跌
  • ✓ RSI > 50: 多方占优，偏多头
  • ✗ RSI < 50: 空方占优，偏空头
  • ⚡ RSI背离: 价格创新高但RSI下降，警惕见顶
  • 参数说明: 14日RSI最常用

布林带 (Bollinger Bands)

  • 上轨: 中轨 + 2倍标准差，压力位
  • 中轨: 20日均线，重要支撑/阻力
  • 下轨: 中轨 - 2倍标准差，支撑位
  • ⚡ 价格触及上轨: 警惕回调风险
  • ⚡ 价格触及下轨: 关注反弹机会
  • ✓ 布林带收口: 波动减小，可能突破
  • ⚡ 布林带开口: 波动加大，趋势确认
  • 正常范围: 约95%的价格应在带内

成交量 (Volume)

  • 定义: 一定时间内的交易数量
  • ✓ 价涨量增: 多方强劲，趋势可能延续
  • ✗ 价跌量增: 空方强劲，可能继续下跌
  • ⚡ 缩量整理: 变盘前兆，观望等待
  • ✓ 放量突破: 突破有效，趋势确认
  • ✗ 缩量突破: 可能是假突破，谨慎
  • 颜色: 绿色=上涨日，红色=下跌日
